CaseReader factory function.

openmdao.recorders.case_reader.CaseReader(filename, pre_load=True)[source]¶

Return a CaseReader for the given file.

Parameters:
filename : str

A path to the recorded file. Currently only sqlite database files recorded via SqliteRecorder are supported.

pre_load : bool

If True, load all the data into memory during initialization.
Returns:
reader : BaseCaseReader

An instance of a CaseReader.
